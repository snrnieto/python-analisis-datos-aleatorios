Menú Dashboard

Este es el menú principal con el que inicia el sitio web

Donde están las opciones:
1.Dashboard
2.Manual de usuario
3.Generar datos

Menú Dashboard

Al momento de ingresar por primera vez a la página, se mostrará un texto "Cargando datos", significa que los datos aleatorios se están generando y recogiendo desde la base de datos.

Generar datos

Para generar los datos podrá configurar diferentes caracteristicas:
1. La primera opción permite escoger entre 6 vectores.
2.La segunda opción permite escoger la distribución con la que se van a generar los datos.
3.La tercera opción permite escoger la naturaleza de los datos(Continua, Discreta).

Opciones de los datos

Con los datos se pueden hacer las siguientes opciones:
1. Ver predicción
2. Ver datos
3. Análisis estadistico
4. Ver comparación

Ver predicción

Se dibujará una linea recta intentando obtener la distancia mínima entre todos los datos

Ver datos

Se mostrarán cada uno de los 1000 datos con su respectivo valor

Ver análisis estadistico

Se abrirá otra ventana mostrando información estadistica (Promedio,Máximo,Mínimo,Desviación,Varianza,Correlación) sobre los datos y también la gráfica de un histograma con respecto a los datos.

Ver comparación

Se mostrará una comparación de los datos reales con los datos que se predijeron.
También se mostrará cual es el error promedio.
